Circle-to-Search Desktop

CircleToSearch-desktop is a high-performance, cross-platform screenshot search tool written in Rust that brings Android's Circle to Search functionality to Desktop. The application captures screen regions and instantly searches them via Google Lens. It delivers a lightweight binary with async processing, supporting Wayland, X11, and multiple desktop environments.

Rust GTK4 Cross-Platform Linux Windows Python
